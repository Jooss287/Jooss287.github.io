Django와 Tomcat 연동하기

Apr 9, 2020 2020-04-09T18:30:00+09:00 by SeongsooJoo
Updated Jul 28 2021-07-28T17:41:41+09:00 3 min

아파치 톰캣을 정상적으로 설치 한 뒤 Django와 연결하기 위해서 WSGI(Web Server Gateway Interface)를 사용하여야 한다. WSGI는 파이썬에서 어플리케이션이 웹 서버와 통신하기 위한 인터페이스로 웹 서버와 웹 어플리케이션 중앙에서 미들웨어로써 동작한다.

1

요청 → 웹서버 → WSGI Server → Django

Django는 mod_wsgi를 사용하여 WSGI Server를 구현한다.

windows+Anaconda+mod_wsgi

mod_wsgi의 윈도우용 버전은 컴파일버전을 더이상 지원하지 않는다.
비공식으로 깃허브 에서 다운받아 컴파일하여 사용한다.

컴파일을 위해 빌드 도구 를 다운받아 설치한다. 빌드도구의 설치가 왼료되지 않으면 mod-wsgi의 설치가 불가능하다.

mod-wsgi의 설치를 위해 홈페이지에서 Release파일을 받는다.
다운받은 압축파일의 압축을 풀고 Anaconda의 가상환경에서 mod_wsgi를 설치한다 압축파일은 c:\download에 있다고 가정한다.

1

(conda vEnv) C:\download> python setup.py install

성공적으로 설치되었으면 mod_wsgi-express module-config를 입력하여 설정파일에 등록할 구문을 확인한다.
